# СИСУ
### Система идентификации сетевых устройств

> СИСУ — это не очередной сетевой сканер.
> Это попытка научить программу понимать, что именно находится в сети.

## О проекте

СИСУ (System for Intelligent Scanning and Understanding) — это проект по интеллектуальному обнаружению и идентификации сетевых устройств.

Проект появился из реальной практической задачи.

Большинство сетевых сканеров умеют находить IP-адреса, MAC-адреса и открытые порты, однако они редко отвечают на главный вопрос:

**Что это за устройство на самом деле?**

СИСУ объединяет информацию из множества независимых источников и сопоставляет полученные данные, чтобы определить тип устройства с максимально возможной достоверностью.

Используются различные методы анализа:

- SNMP
- HTTP
- SSDP / UPnP
- mDNS
- DNS
- TCP Fingerprinting
- Анализ TTL
- База производителей MAC-адресов (OUI)
- Активное исследование устройств
- Система корреляции признаков

Главная задача проекта — не просто просканировать сеть, а понять её.

## Возможности

- интеллектуальная идентификация устройств;
- модульная архитектура;
- механизм корреляции признаков;
- простое расширение новыми модулями;
- база производителей оборудования;
- формирование отчётов;
- ориентация на реальные корпоративные сети.

## Почему СИСУ?

Большинство сетевых сканеров заканчивают работу после обнаружения устройства.

СИСУ делает следующий шаг.

Она пытается определить:

- что это за устройство;
- какая операционная система используется;
- кто производитель;
- это принтер, маршрутизатор, камера, телефон, IoT-устройство или компьютер;
- насколько достоверен полученный результат.

## Статус проекта

Проект активно развивается.

Каждая новая версия расширяет базу сигнатур, улучшает механизм корреляции и повышает точность идентификации устройств.

## Философия проекта

С каждым годом сети становятся всё больше.

Сегодня проблема уже не в том, чтобы обнаружить устройство.

Проблема — понять, что это за устройство.

Именно для решения этой задачи создаётся СИСУ.
